## 🎯 What Makes This Special?

> **Not your ordinary Snake game!** Experience the thrill of strategic combo gameplay that transforms the classic arcade experience into a mind-bending puzzle challenge.

### 🔥 Core Innovation: The Combo System

```
🍎₁ → 🍊₂ → 🍌₃ → 🍇₄ → 🥝₅ = COMBO! 
```

- **5 numbered food blocks** appear simultaneously
- **Eat in sequence (1→2→3→4→5)** for massive combo bonuses
- **Speed increases** with each combo - risk vs reward!
- **Strategic depth** - plan your path or lose your combo

## 🚀 Quick Start

### 🌐 Play Online
Simply visit: **[snake-play.vercel.app](https://snake-play.vercel.app)**

### 💻 Run Locally

```bash
# Clone the repository
git clone https://github.com/kent-leow/snake-game.git

# Navigate to project
cd snake-game

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 🎮 How to Play

<div align="center">

### Controls
**🎯 Arrow Keys** - Move your snake  
**🎲 Strategy** - Plan your path to eat numbers in sequence  
**⚡ Speed** - Each combo makes you faster (and more dangerous!)  

### Winning Strategy
1. **🎯 Plan Ahead** - Look at all 5 numbers before moving
2. **📈 Build Combos** - Sequential eating = bonus points
3. **⚡ Manage Speed** - Faster isn't always better
4. **🧠 Think Spatially** - Position yourself for the next sequence
